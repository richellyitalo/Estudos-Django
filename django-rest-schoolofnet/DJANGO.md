## Sequência Django
* Iniciando projeto:
  * `django-admin startproject NOME_DO_PROJETO`
* Adicionando módulos:
  * `django-admin startapp NOME_DO_MODULO`
* Instalado o *djangorestframework*
  * `pip install djangorestframework`
* Cria as migrations
  * `py manage.py makemigrations`
* Executa a migração
  * `py manage.py migrate`
